# Анализ результатов А/В тестирования

## Цель проекта
Выполнить анализ результатов А/В тестирования: определить, влияет ли изменение цвета фона сайта с белого на чёрный на вовлеченность посетителей.

## Задачи проекта
• Подготовить данные и провести исследовательский анализ  
• Проверить результаты А/В тестирования  

## Используемые библиотеки Python
**pandas | numpy | matplotlib | seaborn | scipy**  

## Результаты исследования
• Различие в конверсии между группами А и В статистически значимо (в пользу группы В)  
• Статистически значимых различий по длительности сессии и глубине просмотра между группами нет  
• Тест следует признать успешным
