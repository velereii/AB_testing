# название

## Цель проекта
Выполнить анализ результатов А/В тестирования: определить, влияет ли изменение цвета фона сайта с белого на чёрный на вовлеченность посетителей.

## Задачи проекта
- Подготовить данные и провести исследовательский анализ  
- Проверить результаты А/В тестирования  

## Описание данных
Датасет содержит информацию о взаимодействиях пользователей с веб-сайтом. Каждый пользователь принадлежит к одной из двух групп — A (фон сайта белый, контрольная) или B (фон сайта чёрный, экспериментальная).

## Используемые библиотеки
pandas numpy matplotlib seaborn scipy  

## Результаты исследования
- Средние показатели просмотров страниц и времени на сайте статистически не различаются в обеих группах  
- Конверсия в целевое действие в группе B практически в три раза выше чем в группе А  
- **Рекомендация:** внедрить вариант **B** как более эффективный.
